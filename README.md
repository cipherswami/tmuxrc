# TMUXRC

A simple and minimal tmux configuration for everyday development. It includes sensible defaults, Vim-style keybindings, Catppuccin theme, and a small set of plugins managed with TPM to provide a clean and efficient terminal experience.

## Requirements

* tmux (3.2 or later)
* Git

## Installation (Linux)

Clone the repository recursively into your tmux configuration directory:

```bash
git clone --recursive https://github.com/cipherswami/tmuxrc.git ~/.config/tmux
```

Start tmux and install the plugins: `<prefix> I`
