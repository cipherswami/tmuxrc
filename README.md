# LazyTMUXRC

A simple and minimal tmux configuration inspired by LazyVim for everyday development. It provides sensible defaults, LazyVim-style keybindings, the Catppuccin theme, and a carefully selected set of TPM-managed plugins to deliver a fast, ergonomic, and cohesive terminal workflow.

## Requirements

* tmux (3.2 or later)
* Git

## Installation (Linux)

Clone the repository recursively into your tmux configuration directory:

```bash
git clone --recursive https://github.com/cipherswami/lazytmuxrc.git ~/.config/tmux
```

Start tmux and install the plugins: `<prefix> I`
